# JournalUp

**JournalUp** is a web application designed to help users document their thoughts, reflect on their daily experiences, and build a consistent journaling habit. Upon logging in, users can post their journals, view past entries, and receive daily affirmations. The platform is user-friendly, motivating, and encourages a positive habit of journaling every day.

## About the Project

**JournalUp** is designed to help users develop the habit of journaling by providing a simple yet effective platform to write down thoughts and experiences. It features a clean and minimalistic design, allowing users to easily post their journal entries, view their history, and get a daily affirmation to inspire them.

The project focuses on helping users reflect on their day, improve their mindset, and stay consistent with journaling.

---

## Features

- **User Authentication**: Secure sign-up and login system for users to access their journal.
- **Post Journals**: Allows users to write, save, and manage their journal entries.
- **View Journal Posts**: Users can browse and read all their previous journal entries.
- **Daily Affirmation**: A motivational affirmation is shown daily to encourage the user to start journaling.
- **Responsive Design**: Optimized for both desktop and mobile devices for a seamless experience across all screen sizes.

## Hosting

As a beginner eager to explore cloud technologies, I hosted this website using **Amazon EC2** to gain hands-on experience with cloud services. This project allowed me to learn about deploying and managing web applications in the cloud. The backend is connected to a MySQL database hosted on **AWS RDS**, providing valuable insights into cloud infrastructure and database management.
